Kürzlich erlangte Informationen von Task Force 313 haben die Existenz von mobilen Erntern bestätigt, die von [[Aztechnology]] hergestellt werden. Dieses neue System, das auf einem von Ares Macrotechnology verwendeten Design basiert, das einfach „das Gerät“ genannt wird, besteht aus einem Kommandofahrzeug/Energietransmitter, einem Erntemodul und einem „Energieversorgungs- und Harmonisierungsmodul“, das fünfzig gefangene Metamenschen in modifizierten Valkyrie-Modulen nutzt. Die mobilen Ernter werden auf speziellen übergroßen Sattelschleppern transportiert und von modifizierten, fünfundzwanzig Meter langen Anhängern aus eingesetzt, die sich zu kleinen Plattformen ausklappen lassen. Die Daten zeigen, dass mobile Ernter bei voller Leistung etwa ein Fünftel der Kapazität und Reichweite ihrer größeren Gegenstücke haben. Sie sind zwar nicht so leistungsfähig wie ein normaler Ernter, aber sie sind doppelt so toxisch und disruptiv für die lokale Manasphäre, weil sie mehr rohe Gewalt einsetzen, um zu funktionieren. In den Notizen einer Technikerin heißt es: „Der normale [[Ernter]] arbeitet mit einem gewissen Maß an Präzision, um das geerntete Mana zu erhalten. Mobile Ernter sollen das Mana hingegen so schnell wie möglich herausreißen und weiterleiten, sodass die Ernte zwar brauchbar ist, aber nur gerade eben so.“ Was mobile Ernter so gefährlich macht, ist ihre Mobilität. Trotz ihrer Größe und ihres Gewichts können sie im Transportmodus fast jedes Gelände aus eigener Kraft durchqueren, und sie lassen sich problemlos mit Schwerfrachtflugzeugen oder Frachtschiffen transportieren. Das bedeutet, dass die disianischen Streitkräfte bei Bedarf kurzfristig ein komplett neues Netzwerk in der Nähe einer Leylinie aufbauen können, um neutralisierte Ernter zu ergänzen … oder einfach eine lokale Manasphäre wie mit einer schmutzigen Bombe zu zerstören. Mobile Ernter haben jedoch eine Schwäche: Sie verfügen nicht über die gleichen Sicherheitsvorkehrungen wie die normalen Ernter. Das bedeutet, dass sie, sobald sie entdeckt werden, mit konventionellen Mitteln ausgeschaltet oder nach ihrer Aktivierung sicher abgeschaltet werden können. Das Haupthindernis besteht jedoch weiterhin darin, sie überhaupt zu finden.
